2. **Word Documents or Other Document Types:**
   - If you're using a document processor like Microsoft Word, you can delete the header section and move all the content (like titles, navigation text) to the footer.
   
   Steps in Microsoft Word:
   - Go to `Layout` -> `Header & Footer`.
   - Select `Remove Header`.
   - In the `Footer`, add your title and other necessary elements.
